# Return Reminder Email

**Trigger:** 1 day before rental end date
**Tool:** Shopify Email, Klaviyo, Booqable automation, or manual send

---

## Subject line

```
Return pickup tomorrow — [Your Brand] · Order [ORDER_NUMBER]
```

---

## Email body

```
Hi [CUSTOMER_NAME],

Your rental period ends tomorrow. We'll be coming to collect your gear — return pickup is free.

RETURN PICKUP
─────────────
Date: [RETURN_DATE]
Time window: [RETURN_WINDOW]
Address: [DELIVERY_ADDRESS]

PLEASE HAVE READY
─────────────────
✓ All rented items
✓ All accessories (poles, stakes, stuff sacks, etc.)
✓ Items in reasonably clean condition (shake out dirt, empty pockets)

WHAT TO EXPECT
──────────────
Our team will inspect each item on-site. Normal trail wear is expected and fine.

If you need to extend your rental, contact us today before your rental end date:
Email: [email@yourbrand.ge]
WhatsApp: [+995 XXX XXX XXX]

Need to change pickup time? Reply to this email or message us on WhatsApp.

Thank you for renting with [Your Brand]!
The [Your Brand] Team
```
