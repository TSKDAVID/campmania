# შეკვეთის დადასტურების ელფოსტა

**Shopify path:** Settings → Notifications → Order confirmation → Edit code

დაამატეთ შემდეგი ბლოკი შეკვეთის ხაზის ელემენტების სექციის შემდეგ.

---

## თემის ხაზი

```
თქვენი გაქირავება დადასტურებულია — [Your Brand] · შეკვეთა {{ order.name }}
```

---

## ელფოსტის ტექსტი (HTML ბლოკი დასამატებლად)

```html
<h2>თქვენი გაქირავება დადასტურებულია</h2>

<p>გამარჯობა {{ customer.first_name }},</p>

<p>გმადლობთ, რომ [Your Brand]-თან დაჯავშნეთ. აი თქვენი გაქირავების დეტალები:</p>

<h3>გაქირავების შეჯამება</h3>
<ul>
  <li><strong>შეკვეთა:</strong> {{ order.name }}</li>
  <li><strong>მიწოდების მისამართი:</strong> {{ shipping_address | format_address }}</li>
  <li><strong>გადახდა:</strong> ნაღდი ანგარიშსწორება მიწოდებისას — გთხოვთ, მოამზადოთ ზუსტი თანხა</li>
</ul>

<h3>მიწოდებისას გადასახდელი თანხა</h3>
<p><strong>{{ order.total_price | money }}</strong> (შეიცავს 5 ლარის მიწოდებას)</p>

<h3>მიწოდების დღემდე</h3>
<ol>
  <li><strong>წაიკითხეთ გაქირავების ხელშეკრულება</strong> — <a href="https://[yourbrand].com/pages/rental-agreement">გაქირავების ხელშეკრულების ნახვა</a></li>
  <li><strong>მოიტანეთ მოქმედი პირადობა</strong> — საქართველოს პირადობის მოწმობა ან პასპორტი</li>
  <li><strong>მოამზადეთ ნაღდი თანხა</strong> — ზუსტი თანხა ზემოთაა მითითებული</li>
  <li><strong>იყავით ხელმისაწვდომი</strong> არჩეულ მიწოდების ფანჯარაში</li>
</ol>

<h3>მიწოდებისას</h3>
<p>ჩვენი გუნდი:</p>
<ul>
  <li>წარგიდგენთ აღჭურვილობას შემოწმებისთვის</li>
  <li>დავადასტურებთ პირადობას (სწრაფი შემოწმება)</li>
  <li>მოგთხოვთ გაქირავების ხელშეკრულებაზე ხელმოწერას</li>
  <li>ვაგროვებთ გადახდას ნაღდად</li>
  <li>გადაგეცემით აღჭურვილობას</li>
</ul>

<h3>დაბრუნების აღება</h3>
<p>დაბრუნების აღება <strong>უფასოა</strong>. გაქირავების დასრულებამდე დაგიკავშირდებით აღების დაგეგმვისთვის თქვენს მისამართზე.</p>

<p>კითხვები? უპასუხეთ ამ ელფოსტას, დაგვიკავშირდით WhatsApp-ზე [+995 XXX XXX XXX]-ზე ან ეწვიეთ <a href="https://[yourbrand].com/pages/contact">ჩვენს საკონტაქტო გვერდს</a>.</p>

<p>ტრასზე გნახავთ,<br>
[Your Brand]-ის გუნდი</p>
```
