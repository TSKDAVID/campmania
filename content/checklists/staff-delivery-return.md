## DELIVERY CHECKLIST

Complete at customer address on delivery day.

### Before leaving warehouse
- [ ] All items pulled and match order
- [ ] Each item passed pre-delivery inspection (see gear-quality checklist)
- [ ] All accessories included (poles, stakes, stuff sacks, etc.)
- [ ] Rental agreement printed (1 copy per order)
- [ ] Order total confirmed: ________ GEL (cash expected)

### At customer address
- [ ] Arrived within scheduled delivery window
- [ ] Customer (or authorized person) present
- [ ] Gear presented for customer inspection
- [ ] Customer satisfied with condition (or issue noted below)
- [ ] **ID verified** — type: [ ] Georgian ID  [ ] Passport
- [ ] ID name matches order name: Y / N
- [ ] ID number recorded: _________________________
- [ ] **Rental agreement signed** by customer
- [ ] **Cash collected:** ________ GEL
- [ ] Change given (if applicable): ________ GEL
- [ ] Security deposit collected (if applicable): ________ GEL
- [ ] Gear handed over to customer
- [ ] Customer briefed on return pickup process
